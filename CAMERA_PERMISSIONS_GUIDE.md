Your video conference application needs camera and microphone access to function properly. Here's how to enable permissions on different devices:

## ðŸ“± **iPhone Safari**

### **Method 1: Through Safari Settings**
1. Open **Settings** on your iPhone
2. Scroll down and tap **Safari**
3. Scroll down to **Camera**
4. Set to **Allow**
5. Scroll down to **Microphone** 
6. Set to **Allow**

### **Method 2: When Prompted**
1. Open Safari and go to your video conference site
2. When you see the permission popup, tap **Allow**
3. If you accidentally tapped "Don't Allow", refresh the page and try again

### **Method 3: Through Site Settings**
1. In Safari, tap the **AA** button in the address bar
2. Tap **Website Settings**
3. Set **Camera** to **Allow**
4. Set **Microphone** to **Allow**

## ðŸ’» **Desktop Browsers**

### **Chrome**
1. Click the **lock icon** in the address bar
2. Set **Camera** to **Allow**
3. Set **Microphone** to **Allow**
4. Refresh the page

### **Firefox**
1. Click the **shield icon** in the address bar
2. Click **Permissions**
3. Set **Use Camera** to **Allow**
4. Set **Use Microphone** to **Allow**

### **Safari (Mac)**
1. Go to **Safari** > **Preferences**
2. Click **Websites** tab
3. Select **Camera** and set to **Allow**
4. Select **Microphone** and set to **Allow**

## ðŸ”§ **Troubleshooting**

### **If Camera Still Doesn't Work:**

1. **Check if another app is using the camera**
   - Close other video apps (Zoom, Teams, FaceTime, etc.)
   - Try refreshing the page

2. **Restart your browser**
   - Close all browser windows
   - Reopen and try again

3. **Check system permissions (Mac)**
   - Go to **System Preferences** > **Security & Privacy** > **Camera**
   - Make sure your browser is checked

4. **Check system permissions (Windows)**
   - Go to **Settings** > **Privacy** > **Camera**
   - Make sure camera access is enabled

### **For iPhone Users:**
- Make sure you're using **Safari** (not Chrome or other browsers)
- iOS requires Safari for proper camera access
- Try refreshing the page after enabling permissions
