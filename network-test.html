üîß Troubleshooting Steps

  1. Check WiFi Network: Ensure both devices are on the same WiFi network
  2. Check IP Address: Verify the server IP is 192.168.1.199
  3. Check Firewall: Make sure macOS firewall allows incoming connections
  4. Try Different Port: If 3000 doesn't work, try 8080 or 5000
  5. Check Router Settings: Some routers block device-to-device communication
